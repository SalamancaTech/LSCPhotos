LSC Photobooth // v2.5LSC Photobooth is a client-side multimodal synthesis engine designed for high-fidelity character consistency and sequential art generation. It leverages the gemini-2.5-flash-image-preview model (Nano Banana) to generate complex compositions, photo strips, and comic pages.Unlike standard text-to-image interfaces, this application utilizes a Composite Assembly Pipeline: rather than generating a multi-panel image in a single inference pass (which often leads to mode collapse or repetition), the engine deconstructs the request into parallel tasks, generates unique panels with enforced variance, and programmatically stitches them into the final layout.üèó System ArchitectureThe Composite PipelineDeconstruction: The user's request (e.g., "4-Panel Comic") is broken down into discrete task objects.Pose Injection: The system injects unique pose/expression constraints into each task's prompt to prevent "static character" syndrome.Parallel Inference: The engine executes concurrent API calls to Gemini 2.5 Flash. This ensures each panel receives the model's full attention regarding character identity.Canvas Stitching: The client-side logic receives the raw Base64 buffers and composites them onto a mathematically generated canvas (Strip, Grid, or Single Page) before rendering the final output.Core ModulesIdentity Binding Protocol: Users can tag reference images (e.g., "Character A"). The system prompt explicitly maps specific visual inputs to semantic actors in the scene description to minimize hallucinations.Correction Pen (Inpainting): A canvas-based editing layer allowing users to draw on generated results and provide natural language instructions for targeted regeneration.Session Lightbox: A volatile memory gallery that persists generated artifacts for the duration of the session.üöÄ Modes of Operation1.
